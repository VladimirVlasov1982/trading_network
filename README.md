# Онлайн платформа торговой сети электроники
***
### Подготовка проекта
Выполните команду: 
```
docker-compose up -d
```

Для начала нужно создать необходимые таблицы в базе данных с помощью команды:
```
python manage.py migrate
```

Запустите сервер:
```
python manage.py runserver
```
